Blackjack Game 🎰♠️

A simple console-based Blackjack game made with Kotlin for our Object-Oriented Programming project.

This project adds extra strategy elements to classic Blackjack through:

💰 Wallet & betting system
🛒 Item shop
🎒 Inventory system
⏱ Deadline/round progression system

Features
Classic Blackjack gameplay against a dealer
Betting system with starting money
Multiple rounds with increasing target amounts
Buy and use special items from the shop
Inventory management

Object-oriented class structure
Special Items
💎 Double or Nothing — Double reward after winning
🛡 Insurance — Protects bet from dealer Blackjack
👁 Peek Card — Reveal dealer hidden card
🔑 Get Out of Jail — Cancel one losing round
